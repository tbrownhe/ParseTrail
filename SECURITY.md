# Security Policy

## Supported Versions

The latest release is supported. Please stay current to receive fixes and security updates.

## Reporting a Vulnerability

If you believe you have found a security issue, email: security@parsetrail.com

Please include:
- A description of the issue and impact
- Steps to reproduce or a proof of concept
- Affected version/commit

We will acknowledge receipt and follow up with remediation details or requests for more information.

## Public Disclosure

Please avoid discussing potential vulnerabilities publicly until we have confirmed and addressed them. Once a fix is available, we will coordinate disclosure details.

Thank you for helping keep ParseTrail secure.
